# Hotel Design & Color Reference

This project is a **luxury hotel website/branding project**. Before making any
design or color-related changes, read the full design philosophy guide at
`HOTEL_DESIGN_GUIDE.md` (in project root).

Key constraints:
- All color decisions must be **strategic**, not decorative — tied to brand
  positioning, guest expectations, and location context.
- Use the **80/20 color rule** and **three-color rule** from the guide.
- Use **solid colors only** — no CSS gradients in backgrounds or overlays.
- Preview themes via the static site in `delete_later/` before committing
  to palette changes in production code.
- The 6 curated theme categories are: Timeless Neutrals, Calming Blues &
  Greens, Earthy & Botanical, Muted Pastels & Pinks, North Star Signature
  (gold + warm brown), and Crimson Warmth.
